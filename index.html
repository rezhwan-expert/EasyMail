EasyMail.NET

Send email in c# with easy method,in just 4 line of code


Download From Nuget

Simple Use

Setup Server Information
- Mail mail = new Mail();
- mail.R_SMTP_Server = "test@smt_or_122.22.example";
- mail.R_SMTP_User = "example@example.com";
- mail.R_SMTP_Password = "your_password_user";
- mail.R_SMTP_Port = 25;default smtp port or smtp@gmail port is 587
- mail.R_HTML_Body = false; set true if you want send html email,by default value is false
- mail.R_SSL_Enable = true; set true,by default value is false
Send Mail
- mail.R_Send_Mail("to", "subject", "msg");
Send Mail With Attachment
- mail.R_Send_Mail("to", "subject", "msg", "AttachMent");

Get Send Status

- bool send = Mail_Status.RIsSend;
- bool notsend = Mail_Status.RIsFailed;
